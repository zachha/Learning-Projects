One of the commonly used logos for JavaScript

Origins of JavaScript

  • JavaScript was originally developed in 1995 under the name Mocha and was officially called LiveScript when it shipped in beta.
  • Due to JavaScript's original inability to process code asyncrhonously, the language was often looked down upon.
  • With the addition of AJAX, JavaScript gained an even larger following and a large party of frameworks and libraries were created.

Javascript joins Back-end Dev

  • JavaScript had been used sparingly in server-side implementations, but in 2009 Node.js was released, which allowed full-stack development using JS.
